# Constellation

Integrative bioinformatics platform for analysis of diverse omic and
bioanalytical datasets. A shared core of physically grounded primitives
feeding domain modules for mass spectrometry, sequencing, structure /
dynamics, codon optimization, and NMR.

- [Architecture](architecture.md) — core vs. domain modules, import DAG, third-party tool pattern.

Constellation is in early scaffold. See the `Ports in flight` section
of `CLAUDE.md` for what's being rewritten from existing lab packages
(Cartographer, NanoporeAnalysis, CoLLAGE, Contour) and the order of
the rewrite.
