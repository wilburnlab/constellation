# docs/

Source for the user-facing documentation site. Currently a minimal
mkdocs-ready skeleton (see `index.md` and `architecture.md`).

To render locally once an mkdocs config lands:

```bash
pip install mkdocs-material
mkdocs serve
```

No `mkdocs.yml` yet — added when there's enough documentation to warrant
the site build. The markdown files are already useful as plain GitHub
docs in the meantime.
